## Getting Started

### 1. Clone the repo

`git clone https://github.com/woowa-techcamp-2022/web-moneybook-1.git`

### 2. Install NPM Package

`npm install`

### 3. .env 생성 및 환경변수 설정

- BASE_URL은 API_END_POINT를 나타냅니다.
  `touch .env`

```
DB_HOST=""
DB_USER=""
DB_PASSWORD=""
DB_NAME=""
BASE_URL=""
```

### 4. build client, start server

```
npm run build
npm run start
```
